Custom Interactions

The Xaml.Behaviors.Interactions.Custom package provides a comprehensive collection of ready-to-use behaviors, actions, and triggers for Avalonia applications. These components cover a wide range of scenarios, from basic UI interactions to complex animations and system integrations.

Installation

Ensure you have the package installed:

<PackageReference Include="Xaml.Behaviors.Interactions.Custom" Version="..." />

Categories

The interactions are organized into the following categories:

  • Actions: General purpose actions.
  • Animations: Behaviors and actions for controlling animations.
  • AutoCompleteBox: Interactions specific to the AutoCompleteBox control.
  • Automation: Behaviors for accessibility and automation properties.
  • Button: Interactions specific to Button controls.
  • Carousel: Interactions for Carousel controls.
  • Collections: Actions for modifying collections.
  • Composition: Advanced composition and visual interactions.
  • ContextDialogs: Behaviors for managing context dialogs.
  • Control: General control behaviors.
  • Converters: Value converters used in interactions.
  • Core: Core infrastructure and base classes.
  • Cursor: Behaviors for managing cursor state.
  • Debugging: Behaviors for debugging and visualization.
  • Dialog: Interactions for showing dialogs.
  • ExecuteCommand: A rich set of behaviors for executing commands in response to various events and gestures.
  • FileUpload: Interactions for file upload scenarios.
  • Focus: Behaviors for managing focus.
  • Gestures: Triggers for touch and pointer gestures.
  • Icon: Interactions for managing icons.
  • Input: Advanced input behaviors.
  • InputElement: Triggers and actions for InputElements.
  • ItemsControl: Interactions for ItemsControls.
  • Layout: Behaviors affecting layout.
  • ListBox: Interactions for ListBox.
  • Notifications: Actions for showing notifications.
  • Popup: Interactions for Popups.
  • RenderTarget: Behaviors for rendering to bitmaps.
  • Screen: Interactions related to screen information.
  • ScrollViewer: Behaviors for ScrollViewer.
  • SelectingItemsControl: Interactions for SelectingItemsControl.
  • Show: Behaviors for showing/hiding elements.
  • SplitView: Interactions for SplitView.
  • System: System and environment interactions.
  • TabControl: Interactions for TabControl.
  • TextBox: Interactions for TextBox.
  • ThemeVariant: Behaviors for managing theme variants.
  • ToolTips: Interactions for ToolTips.
  • Transitions: Behaviors for managing transitions.
  • TreeView: Interactions for TreeView.
  • Validation: Behaviors for input validation.
  • ViewModel: Actions and behaviors for interacting with ViewModels.
  • Window: Interactions for Window management.
  • WriteableBitmap: Behaviors for WriteableBitmap manipulation.
