ParallaxRatio

Gets or sets the parallax ratio. 0.0 means no movement (static). 1.0 means moves with scroll (normal). Values between 0 and 1 create a "far away" depth effect. Negative values move in reverse.

public double ParallaxRatio { get; set; }
